### **Week 2 Project: E-Commerce Product Showcase**

**Project Overview**  
Create an interactive product showcase page for an online store (choose: electronics, fashion, books, or home goods). This project focuses on CSS layout, responsive design, and creating engaging user interfaces for conversion optimization.

**Requirements**  
- **Product grid** using CSS Grid with hover effects
- **Product detail modal** that opens on click
- **Shopping cart sidebar** (static functionality only)
- **Filter/sort functionality** (UI only, no JavaScript logic needed)
- **Responsive product cards** with:
  - Product image
  - Title
  - Price
  - Rating stars
  - Add to cart button
- **Typography hierarchy** for readability
- **Color scheme** aligned with brand identity
- **Micro-interactions** (hover states, transitions, loading animations)
- **Cross-browser compatibility** testing report
- **Performance optimization** (image compression, CSS organization)

**Real-World Features**  
- Mobile-responsive shopping experience
- Accessible color contrast ratios
- Print stylesheet for product pages
- Dark mode implementation
- CSS custom properties for theme consistency

**Deliverables**  
1. Complete product showcase with 6+ products
2. CSS architecture documentation
3. Browser compatibility report
4. Performance audit (Lighthouse scores)
5. Design system documentation (colors, typography, spacing)

**Rubric (30 Points)**  
| **Criteria** | **Professional Standard** | **Points** |
|-------------|--------------------------|------------|
| **Layout & Grid** | Advanced Grid/Flexbox implementation | 6 |
| **Responsive Design** | Mobile → tablet → desktop breakpoints | 6 |
| **UI/UX Quality** | Intuitive interactions, visual hierarchy | 5 |
| **Performance** | Optimized assets, fast loading | 4 |
| **Cross-browser** | Consistent experience across browsers | 4 |
| **Documentation** | Design system, implementation notes | 3 |
| **Code Quality** | Clean, maintainable, organized CSS | 2 |
